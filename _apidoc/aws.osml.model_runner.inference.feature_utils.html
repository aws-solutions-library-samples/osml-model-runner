aws.osml.model_runner.inference.feature_utils.create_mock_feature_collection(payload: BufferedReader, geom=False) → FeatureCollection[source]

This function allows us to emulate what we would expect a model to return to MR, a geojson formatted FeatureCollection. This allows us to bypass using a real model if the NOOP_MODEL_NAME is given as the model name in the image request. This is the same logic used by our current default dummy model to select detection points in our pipeline.

Parameters:

  • payload – BufferedReader = object that holds the data that will be sent to the feature generator

  • geom – Bool = whether or not to return the geom_imcoords field in the geojson

Returns:

FeatureCollection = feature collection containing the center point of a tile given as a detection point
