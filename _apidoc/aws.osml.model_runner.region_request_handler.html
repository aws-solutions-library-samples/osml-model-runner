process_region_request(region_request: RegionRequest, region_request_item: RegionRequestItem, raster_dataset: Dataset, sensor_model: SensorModel | None = None, metrics: MetricsLogger = None) → JobItem[source]

Processes RegionRequest objects that are delegated for processing. Loads the specified region of an image into memory to be processed by tile-workers. If a raster_dataset is not provided directly it will poll the image from the region request.

Parameters:

  • region_request – RegionRequest = the region request

  • region_request_item – RegionRequestItem = the region request to update

  • raster_dataset – gdal.Dataset = the raster dataset containing the region

  • sensor_model – Optional[SensorModel] = the sensor model for this raster dataset

  • metrics – MetricsLogger = the metrics logger to use to report metrics.

JobItem
