aws.osml.model_runner.common.feature_utils.get_feature_image_bounds(feature: Feature) → Tuple[float, float, float, float] | None[source]

This is a utility function that creates a [minx, miny, maxx, maxy] tuple for the boundary of the image geometry of a feature. If no image geometry property can be found None is returned.

Parameters:

feature – the feature to calculate bounds for

Returns:

the bounds or None
