Bases: threading.Thread

A class that represents a thread of control.

This class can be safely subclassed in a limited fashion. There are two ways to specify the activity: by passing a callable object to the constructor, or by overriding the run() method in a subclass.

run() → None

Method representing the thread’s activity.

You may override this method in a subclass. The standard run() method invokes the callable object passed to the object’s constructor as the target argument, if any, with sequential and keyword arguments taken from the args and kwargs arguments, respectively.

model_runner.tile_worker.generate_crops(region: aws.osml.model_runner.common.ImageRegion, chip_size: aws.osml.model_runner.common.ImageDimensions, overlap: aws.osml.model_runner.common.ImageDimensions) → List[aws.osml.model_runner.common.ImageRegion]

Yields a list of overlapping chip bounding boxes for the given region. Chips will start in the upper left corner of the region (i.e. region[0][0], region[0][1]) and will be spaced such that they have the specified horizontal and vertical overlap.

Parameters:

  • region – ImageDimensions = a tuple for the bounding box of the region ((ul_r, ul_c), (width, height))

  • chip_size – ImageDimensions = a tuple for the chip dimensions (width, height)

  • overlap – ImageDimensions = a tuple for the overlap (width, height)

Returns:

List[ImageRegion] = an iterable list of tuples for the chip bounding boxes [((ul_r, ul_c), (w, h)), …]
