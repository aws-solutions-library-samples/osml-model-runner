class model_runner.sink.S3Sink(bucket: str, prefix: str, assumed_role: str | None = None)

Bases: model_runner.sink.sink.Sink

A sink for writing aggregated GeoJSON feature collections to an S3 bucket.

This class handles uploading GeoJSON data to a specified S3 bucket, optionally using an assumed IAM role for accessing the bucket. It validates the bucket’s accessibility and writes the aggregated features to S3.

Parameters:

  • bucket – The name of the S3 bucket.

  • prefix – The prefix within the bucket where the files will be stored.

  • assumed_role – Optional IAM role ARN to assume for accessing the bucket.

Write aggregated GeoJSON feature collection to the S3 bucket.

Validates if the S3 bucket is accessible and uploads a temporary file containing the aggregated features. The object key is derived from the image_id.

  • features – A list of GeoJSON features to be aggregated and written to S3.
