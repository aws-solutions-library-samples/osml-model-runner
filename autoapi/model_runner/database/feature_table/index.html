class model_runner.database.feature_table.DDBItem

DDBItem is a dataclass meant to represent a single item in a DynamoDB table via a key-value pair

The data schema is defined as follows: key: str = the table key to index on value: str = the value of the item (given a key) to use range_key: str = the sort key to index on range_value: str = the value of the item (given a key + range key) to use
