class model_runner.app_config.ServiceConfig

ServiceConfig is a dataclass meant to house the high-level configuration settings required for Model Runner to operate that are provided through ENV variables. Note that required env parameters are enforced by the implied schema validation as os.environ[] is used to fetch the values. Optional parameters are fetched using, os.getenv(), which returns None.

class model_runner.app_config.BotoConfig

BotoConfig is a dataclass meant to vend our application the set of boto client configurations required for OSML

The data schema is defined as follows: default: (Config) the standard boto client configuration sagemaker: (Config) the sagemaker specific boto client configuration
